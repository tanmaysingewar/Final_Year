Max-Heap and Palindrome

A max-heap is a complete binary tree where each node is greater than or equal to its children. In a max-heap, the value of each node is greater than or equal to its children. This property ensures that the maximum element is always at the root of the heap.

Checking if a string of length n is a palindrome can be done in O(n) time complexity. One approach is to use a two-pointer technique, where one pointer starts from the beginning of the string and the other pointer starts from the end of the string. The algorithm compares characters at both pointers and moves the pointers towards the center of the string. If all characters match, the string is a palindrome.
