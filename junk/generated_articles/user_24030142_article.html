What is the time complexity of searching for an element in an unsorted array?

The time complexity of searching for an element in an unsorted array is O(n), where n is the size of the array. This is because, in the worst-case scenario, we need to iterate through every element in the array to find the desired element.

Which of the following operations is performed more efficiently by a doubly linked list than by a singly linked list?

A doubly linked list outperforms a singly linked list in inserting or deleting nodes at the end of the list. This is because a doubly linked list has references to both the previous and next nodes, allowing for efficient insertion and deletion at the end of the list.

Which of the following is a characteristic of a binary search tree?

A binary search tree is a data structure in which each node has at most two children (i.e., left and right children). A characteristic of a binary search tree is that the value of each node is greater than or equal to the values of its left child and less than or equal to the values of its right child.
