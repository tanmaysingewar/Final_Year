String Manipulation in C++

In C++, strings can be manipulated using various methods and functions. The `std::string` class provides an extensive range of functions for string operations, such as retrieving substrings, finding substrings, and replacing substrings. In this article, we will focus on the `substr` and `find` functions.

Substring Retrieval

The `substr` function is used to retrieve a substring from a given string. It takes two parameters: the starting position and the length of the substring. The syntax is as follows:

string substr(size_type pos, size_type n);

where `pos` is the starting position and `n` is the length of the substring.

Substring Replacement

The `find` function is used to find a substring within a given string. It returns the position of the first occurrence of the substring. The syntax is as follows:

size_type find(const string& str, size_type pos = 0);

where `str` is the substring to find and `pos` is the starting position of the search.

    if (pos != std::string::npos) {

Question 3: Finding the Lone Integer

The problem statement is as follows: given an array of integers where each integer appears twice except for one, find the integer that appears only once.

To solve this problem, we can use the concept of bitwise operations. Here's a possible solution:

int findLoneInteger(int arr[], int n) {
    int result = 0;
    for (int i = 0; i < n; i++) {
        result ^= arr[i];
    }
    return result;
}

This solution works by using the XOR operation (^) to find the lone integer. Since each integer appears twice except for one, the XOR operation will cancel out all the duplicate integers, leaving only the lone integer.
